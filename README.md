# JSpider

#### 介绍
在浏览器端使用 JS 快速爬取文件的框架

#### 软件架构
JSpider 分为 
- 请求模块（Ajax）
- HTML文本解析模块（HTMLParser）
- XML文本解析模块（XMLParser）
- 批量下载模块（Downloader）
